Documentación de Full Stack

Documentación creada por el equipo de desarrollo Full Stack, en el que se han usado las siguientes tecnologías:

Uso de librerias Back-End

  • MongoDB - para crear nuestra base de datos.
  • Mongoose – para modelar los datos dentro de la base de datos.
  • Express – nuestro entorno de trabajo web para node.
  • Puppeteer – nos ayuda para hacer scraping.
  • Nodemon – nos ayuda a que el servidor se reinicie automáticamente.
  • Inspector – interfaz de debugging.
  • Socket.io – nos ayuda a crear la aplicación de chat dentro del sitio web.
  • Cors – nos ayuda a usar middleware de Express.
  • React-to-print - nos ayuda a permitir al usuario imprimir su factura
  • Bcrypt – nos ayuda a enctriptar la información del password.
  • Stripe – nos ayuda a aceptar pagos electrónicos dentro del sitio web.
  • Nodemailer – nos ayuda a enviar por correo electrónico la factura de la compra.
  • JSDoc - nos ayuda a generar la documentación dentro del código

Uso de librerías Front-End

  • Socket.io Client – nos ayuda a implementar el chat.
  • React scroll to bottom - nos ayuda a poder hacer scroll en la ventana del chat.
  • Leaflet – librería para usar mapas, nos ayuda a desplegar ubicación de puntos de entrega.
  • Bootstrap – librería para agregar estilo de diseño al sitio web.
  • Axios – nos ayuda a enviar la petición y respuesta http en formato JSON.
  • AntD – nos ayuda con el diseño del sitio web.
  • React PDF-renderer – nos ayuda a crear la factura en PDF.
